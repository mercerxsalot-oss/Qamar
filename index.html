// SETUP GUIDE

01
Add a web server to your Replit bot so it has a URL to ping. For Python (discord.py / any bot):

from flask import Flask
from threading import Thread
app = Flask('')
@app.route('/')
def home(): return "Bot alive!"
Thread(target=lambda: app.run(host='0.0.0.0',port=8080)).start()
02
For Node.js bots add: require('http').createServer((_,r)=>r.end('alive')).listen(3000) — Replit will expose it as a URL automatically.
03
Copy your Replit project's Webview URL (looks like https://name.username.repl.co) and paste it above. Click SAVE URL to remember it across sessions.
04
Set interval to 5 minutes and click START. Your settings persist across page refreshes — monitoring auto-resumes if you had it running.
05
For true 24/7 even when your PC is off: host this HTML on GitHub Pages for free, open it on a phone/tablet that stays plugged in, or use UptimeRobot.com (free tier) pointing to the same Replit URL.
