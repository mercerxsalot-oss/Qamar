// How To Use

01
Add a simple HTTP server to your bot. In Python: from flask import Flask; app = Flask(__name__); @app.route('/'); run(host='0.0.0.0', port=8080) — this gives Replit a URL to serve.
02
Copy your Replit project's public URL — it looks like https://name.username.repl.co — and paste it above.
03
Set the ping interval (5 minutes recommended) and click START. Keep this tab open in your browser. Every ping resets Replit's sleep timer.
04
For true 24/7 uptime even when your browser is closed, host this HTML file on Vercel or GitHub Pages and open it from another always-on device.
